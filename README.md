## Tests
Install [Truffle](https://truffleframework.com):

    npm install -g truffle
    
Install and run [Ganache CLI](https://github.com/trufflesuite/ganache-cli). Keep it running:

    npm install -g ganache-cli
    ganache-cli

Run tests:

    truffle test
